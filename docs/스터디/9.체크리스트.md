# 9. 정리: MCPHUB 구축 시 핵심 체크리스트

---

## 9.1 게이트웨이 역할

- [ ] 단일 진입점(Endpoint) 제공
- [ ] SSO 기반 인증 처리
- [ ] 사용자 정보 헤더 주입 (x-user-id, x-user-role)
- [ ] Personal Key 프록시 (External MCP용)
- [ ] 요청/응답 로깅

---

## 9.2 에러 처리

- [ ] JSON-RPC 에러 구조 그대로 전달 (꿀꺽 금지)
- [ ] `error.data` 필드에 액션 가능한 정보 포함
- [ ] 타임아웃 발생 시 명확한 에러 메시지

---

## 9.3 인증/권한

- [ ] SSO 토큰 검증
- [ ] 도구별 권한 체크 (Pre-filter 또는 Post-check)
- [ ] 민감정보 접근 승인 프로세스

---

## 9.4 운영

- [ ] 등록 시 헬스체크 (initialize, tools/list)
- [ ] 주기적 헬스체크 + 상태 관리
- [ ] 요청/응답 로그 수집
- [ ] 모니터링 대시보드

- [ ] 요청 취소 처리
