# 1. MCP란 무엇인가?

[← 목차로 돌아가기](./0.목차.md)

---

## 1.1 한 줄 정의

**MCP(Model Context Protocol)**는 AI 에이전트(Claude, GPT 등)가 외부 도구나 데이터에 접근할 수 있게 해주는 **표준화된 통신 규약**입니다.

## 1.2 왜 MCP가 필요한가?

기존에는 AI에게 외부 기능을 제공하려면:
- 각 AI 플랫폼마다 다른 방식으로 연동 (OpenAI Function Calling, Claude Tools 등)
- 플랫폼이 바뀌면 코드 재작성 필요
- 도구 간 재사용 불가

MCP는 이를 표준화합니다:

```
┌─────────────────────────────────────────────────────────────────┐
│                        AS-IS (비표준)                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Claude ─── (Claude 전용 API) ─── Jira 도구                     │
│   GPT ───── (OpenAI Function) ─── Jira 도구 (또 다른 코드!)      │
│   Cursor ── (별도 방식) ───────── Jira 도구 (또 다른 코드!!)      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                        TO-BE (MCP 표준)                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Claude ──┐                                                    │
│   GPT ─────┼── (MCP 프로토콜) ─── MCP 서버 (Jira 도구)            │
│   Cursor ──┘                                                    │
│                                                                 │
│   → 하나의 MCP 서버가 모든 AI 클라이언트에서 동작!               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 1.3 MCP의 구성 요소

| 구성 요소 | 역할 | 예시 |
|-----------|------|------|
| **Host** | AI 애플리케이션 (사용자 인터페이스) | Claude Desktop, Cursor IDE |
| **Client** | MCP 프로토콜을 구현한 연결 관리자 | Host 내부에 포함됨 |
| **Server** | 외부 도구/데이터를 제공하는 서버 | Jira MCP, Slack MCP, 사내 API MCP |

```
┌─────────────────────────────────────────────────────────────────┐
│                    MCP 아키텍처 개요                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌───────────────────────────────────────────┐                 │
│   │              HOST (Claude Desktop)        │                 │
│   │  ┌─────────────────────────────────────┐  │                 │
│   │  │            AI/LLM                   │  │                 │
│   │  └─────────────────────────────────────┘  │                 │
│   │                    │                      │                 │
│   │  ┌─────────────────┼─────────────────┐    │                 │
│   │  │     MCP CLIENT  │                 │    │                 │
│   │  │  ┌──────────┐ ┌──────────┐        │    │                 │
│   │  │  │ Client A │ │ Client B │ ...    │    │                 │
│   │  │  └────┬─────┘ └────┬─────┘        │    │                 │
│   │  └───────┼────────────┼──────────────┘    │                 │
│   └──────────┼────────────┼───────────────────┘                 │
│              │            │                                     │
│   ┌──────────▼────┐  ┌────▼──────────┐                          │
│   │  MCP Server A │  │  MCP Server B │                          │
│   │   (Jira)      │  │   (Slack)     │                          │
│   └───────────────┘  └───────────────┘                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
